Methods

(inner) sendRequest(method, requestURL, body, headers) → {Promise}

Sends an HTTP request using XMLHttpRequest.
Parameters:
Name Type Description
method string The HTTP method for the request (e.g., "GET", "POST", "PUT", "DELETE").
requestURL string The URL to send the request to.
body Object | null The request body, if applicable (default is null).
headers Object The request headers as key-value pairs (default is an empty object).
Source:
  • apiUtils.js, line 7
Returns:
A promise that resolves with the response data or rejects with an error.
Type
Promise
Example
const requestOptions = {
  method: 'GET',
  url: 'https://api.example.com/data',
  headers: {
    'Authorization': 'Bearer YOUR_ACCESS_TOKEN'
  }
};
sendRequest(requestOptions.method, requestOptions.url, null, requestOptions.headers)
  .then(response => console.log(response))
  .catch(error => console.error(error));
